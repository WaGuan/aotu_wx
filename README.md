# aotu_wx
凹凸实验室公众号

## 开始

```bash
git clone https://github.com/o2team/aotu_wx.git
cd aotu_wx && npm i -d
```

将`config/config.js.default`重命名为`config/config.js`，并填入`token`/`appid`/`secret`等信息，然后启动：

```bash
npm start
```
